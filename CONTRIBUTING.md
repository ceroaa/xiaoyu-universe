# CONTRIBUTING.md

## Welcome to Contributors
Welcome to the xiaoyu-universe project! We appreciate your interest in contributing. This file provides guidelines for both research collaboration and code contributions to ensure a smooth workflow.

## Research Collaboration Guidelines
To foster a productive research environment, participants are encouraged to follow these collaboration guidelines. Respect, openness, and integrity are paramount in all collaborations.

## Getting Started with Development
To start contributing to our project, please ensure you have the following installed:
- Git
- A suitable code editor (e.g., VSCode, PyCharm)

Follow instructions in the [Getting Started](link-to-getting-started-guide) guide for detailed steps.

## Setting up Research Environment
For researchers, the following tools/resources are recommended:
- Python 3.x
- Jupyter Notebook
- Required libraries listed in the requirements.txt file.

To set up your environment, please use the following commands:

```bash
# Clone the repository
git clone https://github.com/ceroaa/xiaoyu-universe.git
cd xiaoyu-universe

# Install dependencies
pip install -r requirements.txt
```

## Writing and Submitting Research
When writing your research papers, ensure to follow the project’s publication standards. Use the template provided in the repository and follow the submission guidelines outlined in the [Publishing Guide](link-to-publishing-guide).

## Code Contributions
To contribute code:
1. Fork the repository.
2. Create a new branch for your feature.
3. Commit your changes.
4. Push the branch and open a pull request.

## Documentation Guidelines
Documentation should be clear and concise. Follow the existing formatting and structure within documentation files. Always include examples wherever applicable.

## Reporting Issues
If you encounter an issue, please report it on the [Issues Tracker](link-to-issues) with a detailed description and steps to reproduce.

## Academic Citation Requirements
For academics contributing to the project, please cite any relevant work using the predefined citation style provided in the [Citation Guide](link-to-citation-guide).

## Code of Conduct
We expect all participants to adhere to our code of conduct. Please be respectful and inclusive to create a welcoming environment.

## Collaboration Agreement
All contributions should be made with a signed collaboration agreement. Ensure to read through this document prior to making contributions.
